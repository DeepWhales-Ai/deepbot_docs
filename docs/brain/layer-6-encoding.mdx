<H2 id="not-an-api">Bobby has no public API for trading bots</H2>
<Para>
  No raw contract addresses on a webhook. No RSS. No public real time feed. Trading-grade
  Bobby intelligence is structured for one downstream consumer's tool call schema and that
  consumer is Dale.
</Para>
<Para>
  This is not a technical accident. It is a product decision. Selective interoperability,
  not technical complexity, is what protects this. Anyone could build the API. Nobody else
  has the network to put through it.
</Para>

<H2 id="why-it-matters">Why it matters to investors</H2>
<Para>
