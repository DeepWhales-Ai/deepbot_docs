<H2 id="creating">Creating wallets</H2>
<Para>
  Tell Dale which chain and what to call it. <Mono>create me a Base wallet called Sniper</Mono>{' '}
  provisions a fresh wallet on Base with that label. Same shape for Ethereum, BSC,
  Base, Polygon, and Solana. Default chain assignment follows the chain you
  named.
</Para>

<ChatBlock>
  <ChatBubble from="user">create me a base wallet called Sniper</ChatBubble>
  <ChatBubble from="dale">Created. Sniper · Base · 0x4ab...77e. Default chain set to Base for this wallet. <Lime>Fund to start trading →</Lime></ChatBubble>
</ChatBlock>

<H2 id="importing">Importing private keys</H2>
<Para>
  Send Dale the key in a private message with a label.{' '}
  <Mono>import this key as my Trading wallet</Mono> and paste below. Dale detects
  chain from key format automatically. EVM secp256k1 and Solana Ed25519 are both
  supported. The key is encrypted on receipt and the original message{' '}
  <Lime>auto deletes from chat</Lime> after processing.
</Para>

<H2 id="default">Default wallet switching</H2>
<Para>
  <Mono>use my Trading wallet by default</Mono>. Dale switches the active wallet
  for trades and sends without needing per message specs. Switch back at any
  time. The default applies to the chain the wallet sits on.
</Para>

<H2 id="export">Private key export</H2>
<Para>
  Export runs through <Mono>@walletdeepbot</Mono>, not through the Dale chat. Two
  step confirm. The key displays once and{' '}
  <Lime>auto deletes from the chat</Lime> after display. The sensitive number
  never sits in your message history.
</Para>
