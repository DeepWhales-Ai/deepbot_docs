<H2 id="was">What it was</H2>
<Para>
  Smart Wallets tracked a curated set of profitable trader wallets. When one
  of them bought, Shotgun fired.
</Para>

<H2 id="becomes">What it becomes</H2>
<Para>
  The convergence engine reads smart wallet activity as one of several
  weighted source types. Instead of a single wallet triggering a buy,
  convergence requires multiple independent sources to agree before a signal
  fires.
</Para>
<Para>
  This is a <Lime>quieter, harder signal</Lime>. Fewer triggers, higher
  quality. The trade off is what convergence is for.
</Para>
