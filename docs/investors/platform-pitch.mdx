<H2 id="thesis">The thesis</H2>
<Para>
  Crypto's two open problems are intelligence and execution. The intelligence layer is too
  noisy for retail. The execution layer is too fragmented for everyone. <DeepBot /> collapses
  both into one platform with one revenue model.
</Para>

<H2 id="moat">The moat</H2>
<Para>
  Convergence detection is hard to copy. It requires the presence the network has built,
  time series data across cycles, and an encoding layer that prevents third party
  extraction. Each component takes years. We have built all three.
</Para>
